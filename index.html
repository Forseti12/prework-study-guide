CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When setting the colour of something you can type a colour name if its somethign basic or put in the RGB code for the colour.
  • You can put in either a percent of space or a set amount of pixels.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • You need to do a git pull after you merge a branch to main for it to not delete everything.
